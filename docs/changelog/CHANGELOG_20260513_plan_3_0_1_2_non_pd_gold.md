## Notes / lessons

- **Don't parallel-deploy DDL with in-flight MV creation.** The 0.1.2 deploy ran concurrent with the gold-MV background task and added 50 new tables, pushing the metastore past 500 and aborting the MV task with 21 left to create. Sequencing would have avoided the loss. Recoverable by background-retrying the missing MVs later.
- **`EXECUTE IMMEDIATE` quote escapes are non-standard.** `''` doesn't unescape inside the string body; `chr(39)` is the working form. Documented in the rewritten gate 03 file.
